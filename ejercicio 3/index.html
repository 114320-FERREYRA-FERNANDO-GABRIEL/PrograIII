HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y diseñar páginas web. Consiste en una serie de elementos (etiquetas) que definen la estructura y el contenido de una página web. HTML utiliza etiquetas para organizar el contenido en diferentes secciones, como encabezados, párrafos, listas, enlaces, imágenes, formularios, entre otros.
